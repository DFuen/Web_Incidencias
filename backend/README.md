# Backend — Spring Boot 3

Este módulo contendrá la API REST desarrollada con Spring Boot 3 para la gestión de incidencias.

## Pasos iniciales sugeridos

1. Crear un proyecto Spring Boot 3 (Maven o Gradle)
2. Añadir dependencias: Spring Web, Spring Data JPA, Spring Security, PostgreSQL Driver, Lombok, (opcional: Spring Boot DevTools, Spring Validation, Spring Boot Actuator)
3. Configurar la conexión a PostgreSQL en `application.properties` o `application.yml`
4. Implementar el modelo de datos según `requisitos-incidencias.md`
5. Crear controladores, servicios y repositorios para la gestión de incidencias, usuarios, ubicaciones y categorías
6. Implementar autenticación y autorización por roles
7. Permitir subida de archivos (fotos) y servirlos desde el backend
8. Añadir paginación, filtros y endpoints para métricas

Consulta el README principal para más detalles y requisitos.
